方法

    
.set() 設定所有物件的預設屬性  
.bringForward() 上移一層z-index  
.sendBackwards() 後移一層 z-index  
.bringToFront() 最上層 z-index  
.sendToBack() 最底層 z-index  
     
.animate() 動畫  
.center() canvas 正中心 (使用後須使用 setCoords() 將物件控制項一起移動到圖形上)  
.setCoords() 設定座標
這個常用在我們變更物件位置時，控制項並未跟著改變		  
.setControlsVisibility()
  • 哪些編輯把手要顯示 
    setControlsVisibility({
    bl: true, // 左下
    br: true, // 右下
    mb: false, // 下中
    ml: true, // 中左
    mr: true, // 中右
    mt: true, // 上中
    tl: true, // 上左
    tr: false, // 上右
    mtr: true // 旋轉控制鍵
